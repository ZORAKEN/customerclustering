# customerclustering
Project Overview
This project applies unsupervised machine learning techniques to segment customers based on their demographic information, purchasing behavior, and marketing campaign responses. The goal is to identify distinct customer groups that can be used for targeted marketing strategies and business decision-making.


What made it different
Principal Component Analysis (PCA) was applied to reduce dimensionality while preserving 95% of the dataset variance.
The optimal number of principal components was determined using the cumulative explained variance.(cumsum)


Finding the Optimal Number of Clusters
Two evaluation methods were used:
Elbow Method
Silhouette Score
Silhouette diagrams were also generated to visually evaluate cluster quality.

Data Visualization
